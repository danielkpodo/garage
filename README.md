# A Garage Landing Page

This landing page consists of the follwoing features

- Our Products
- About Us
- Pricing
- Team
- Testimonials

**Client:** React, MaterializeCss

## Run Locally

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
